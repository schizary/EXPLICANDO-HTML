HTML Styles

Estilos HTML

O atributo HTML styleé usado para adicionar estilos a um elemento, como cor, fonte, tamanho e muito mais.

<p>I am normal</p>
<p style="color:red;">I am red</p>
<p style="color:blue;">I am blue</p>
<p style="font-size:50px;">I am big</p>

na pratica

I am normal

I am red

I am blue

I am big

O atributo de estilo HTML

Definir o estilo de um elemento HTML pode ser feito com o styleatributo. O atributo HTML stylepossui a seguinte sintaxe:

"<tagname style="property:value;"></tagname>"


A propriedade é uma propriedade CSS. O valor é um valor CSS.

Cor de fundo

A background-colorpropriedade CSS define a cor de fundo de um elemento HTML.

Defina a cor de fundo de uma página como azul claro:

<body style="background-color:powderblue;">
    <h1>This is a heading</h1>
    <p>This is a paragraph.</p>
</body>
            
Defina a cor de fundo para dois elementos diferentes:

<body>
    <h1 style="background-color:powderblue;">This is a heading</h1>
    <p style="background-color:tomato;">This is a paragraph.</p>
</body>

Cor do texto

A colorpropriedade CSS define a cor do texto para um elemento HTML:

<h1 style="color:blue;">This is a heading</h1>
<p style="color:red;">This is a paragraph.</p>

na pratica

This is a heading

This is a paragraph.

Fontes

A propriedade CSS font-familydefine a fonte a ser usada para um elemento HTML:

<h1 style="font-family:verdana;">This is a heading</h1>
<p style="font-family:courier;">This is a paragraph.</p>

na pratica

This is a heading

This is a paragraph.

Tamanho do texto

A font-sizepropriedade CSS define o tamanho do texto para um elemento HTML:

<h1 style="font-size:300%;">This is a heading</h1>
<p style="font-size:160%;">This is a paragraph.</p>

na pratica

This is a heading

This is a paragraph.

Alinhamento de Texto

A text-alignpropriedade CSS define o alinhamento horizontal do texto para um elemento HTML:

<h1 style="text-align:center;">Centered Heading</h1>
<p style="text-align:center;">Centered paragraph.</p>

na pratica

Centered Heading

Centered paragraph.
