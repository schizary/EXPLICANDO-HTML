HTML Div

Elemento HTML Div

O <div>elemento é usado como contêiner para outros elementos HTML.

O <div>elemento é por padrão um elemento de bloco, o que significa que ocupa toda a largura disponível e vem com quebras de linha antes e depois.

Exemplo

Lorem Ipsum
I am a div
dolor sit amet.
Lorem Ipsum <div>I am a div</div> dolor sit amet.

<div> como um contêiner

O <div>elemento é frequentemente usado para agrupar seções de uma página da web.

Exemplo

Um elemento <div> com elementos HTML:

London

London is the capital city of England.

London has over 13 million inhabitants.

<div>
    <h2>London</h2>
    <p>London is the capital city of England.</p>
    <p>London has over 13 million inhabitants.</p>
</div>

Alinhar ao centro um elemento <div>

Se você tiver um <div> elemento que não tenha 100% de largura e quiser alinhá-lo centralmente, defina a marginpropriedade CSS como auto.

Exemplo

<style>
    div {
        width:300px;
        margin:auto;
    }
</style>

Resultado:

London

London is the capital city of England.

London has over 13 million inhabitants.

Alinhando elementos <div> lado a lado

Ao construir páginas da web, muitas vezes você deseja ter dois ou mais <div>elementos lado a lado.

Existem diferentes métodos para alinhar elementos lado a lado, todos incluem alguns estilos CSS. Veremos os métodos mais comuns:

Flutuador

A floatpropriedade CSS não foi originalmente criada para alinhar <div> elementos lado a lado, mas tem sido usada para esse propósito há muitos anos.

A propriedade CSS floaté usada para posicionar e formatar conteúdo e permite que os elementos flutuem próximos uns dos outros em vez de uns sobre os outros.

Bloco embutido

Se você alterar a propriedade <div>do elemento de para , os elementos não adicionarão mais uma quebra de linha antes e depois e serão exibidos lado a lado em vez de um em cima do outro.displayblockinline-block<div>

Exemplo

Como usar display: inline-block para alinhar elementos div lado a lado:

<style>
    div {
    width: 30%;
    display: inline-block;
}
</style>

Flexível

O CSS Flexbox Layout Module foi introduzido para facilitar o design de uma estrutura de layout responsiva e flexível sem usar float ou posicionamento. Para fazer o método CSS flex funcionar, envolva os <div>elementos com outro <div>elemento e dê a ele o status de flex container.

Exemplo

Como usar o flex para alinhas elementos div lado a lado

<style>
    .mycontainer {
        display: flex;
    }
    .mycontainer > div {
        width:33%;
    }
</style>

Grade

O Módulo CSS Grid Layout oferece um sistema de layout baseado em grade, com linhas e colunas, facilitando o design de páginas web sem a necessidade de usar flutuadores e posicionamento.
Parece quase igual a flex, mas tem a capacidade de definir mais de uma linha e posicionar cada linha individualmente.
O método de grade CSS requer que você envolva os <div>elementos com outro <div>elemento e forneça o status de contêiner de grade, e você deve especificar a largura de cada coluna.

Exemplo

Como usar a grade para alinhar elementos <div> lado a lado:

<style>
.grid-container {
    display: grid;
    grid-template-columns: 33% 33% 33%;
}
</style>
