HTML Lists

Listas HTML

As listas HTML permitem que os desenvolvedores da web agrupem um conjunto de itens relacionados em listas.

Exemplo

Lista ordenada

  • Coffee
  • Tea
  • Milk

Lista não-ordenada

  1. Coffee
  2. Tea
  3. Milk

Lista HTML não ordenada

Uma lista não ordenada começa com a <ul>tag. Cada item da lista começa com a <li> tag. Os itens da lista serão marcados com marcadores (pequenos círculos pretos) por padrão:

  1. Coffee
  2. Tea
  3. Milk
                    

<ul>
    <li>Coffee</li>
    <li>Tea</li>
    <li>Milk</li>
</ul>

Listas de descrição HTML

HTML também oferece suporte a listas de descrição. Uma lista de descrição é uma lista de termos, com uma descrição de cada termo. A <dl>tag define a lista de descrição, a <dt>tag define o termo (nome) e a <dd> tag descreve cada termo:

Exemplo

Coffee
- black hot drink
Milk
- white cold drink
<dl>
    <dt>Coffee</dt>
    <dd>- black hot drink</dd>
    <dt>Milk</dt>
    <dd>- white cold drink</dd>
</dl>
