HTML Formatting

Formatação de texto HTML

HTML contém vários elementos para definir texto com um significado especial.

Exemplo

This text is bold


This text is italic


This is subscript and superscript


Elementos de formatação HTML

Os elementos de formatação foram projetados para exibir tipos especiais de texto:

<b>- Texto em negrito
<strong>- Texto importante
<i>- Texto em itálico
<em>- Texto enfatizado
<mark>- Texto marcado
<small>- Texto menor
<del>- Texto excluído
<ins>- Texto inserido
<sub>- Texto subscrito
<sup>- Texto sobrescrito

Elementos HTML e <strong>

O <b> elemento HTML define texto em negrito, sem qualquer importância extra.

<b>This text is bold</b>

O elemento HTML <strong> define texto com grande importância. O conteúdo interno 
normalmente é exibido em negrito.

<strong>This text is important!</strong>

Elementos HTML <i> e <em>

O elemento HTML <i>define uma parte do texto com uma voz ou humor alternativo. O conteúdo interno normalmente é exibido em
itálico. Dica: A <i> tag costuma ser usada para indicar um termo técnico, uma frase de outro idioma, um pensamento, o nome de um navio, etc.

<i>This text is italic</i>
This text is italic

O elemento HTML <em> define o texto enfatizado. 
O conteúdo interno normalmente é exibido em itálico.
Dica: um leitor de tela pronunciará as palavras <em> com ênfase, usando ênfase verbal.

<em>This text is emphasized</em>
This text is emphasized

Elemento HTML <small>

O elemento HTML <small> define um texto menor:

<small>This is some smaller text.</small>
This is some smaller text.

Elemento HTML <mark>

O elemento HTML <mark> define o texto que deve ser marcado ou destacado:

<p>Do not forget to buy <mark>milk</mark> today.</p>

Do not forget to buy milk today.

Elemento HTML <del>

O elemento HTML <del> define o texto que foi excluído de um documento. Os navegadores geralmente traçam uma linha no texto excluído:

<p>My favorite color is <del>blue</del> red.</p>

My favorite color is blue red.

Elemento HTML <ins>

O elemento HTML <ins>define um texto que foi inserido em um documento. Os navegadores geralmente sublinharão o texto inserido:

<p>My favorite color is <del>blue</del> <ins>red</ins>.


My favorite color is blue red.

Elemento <sub> HTML

O elemento HTML <sub> define o texto subscrito. O texto subscrito aparece meio caractere abaixo da linha normal e às vezes é renderizado em uma fonte menor. O texto subscrito pode ser usado para fórmulas químicas, como H 2 O:

<p>This is <sub>subscripted</sub> text.</p>

This is subscripted text.

Elemento HTML <sup>

O elemento HTML <sup>define texto sobrescrito. O texto sobrescrito aparece meio caractere acima da linha normal e às vezes é renderizado em uma fonte menor. Texto sobrescrito pode ser usado para notas de rodapé, como WWW [1] :

<p>This is <sup>superscripted</sup> text.</p>

This is superscripted text.
